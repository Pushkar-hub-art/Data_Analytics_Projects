# IPL 2025 Data Analysis

A Python project analyzing IPL 2025 batting and bowling stats using pandas, numpy, and matplotlib.

## Features
- Best/worst batter and bowler by various metrics (runs, average, strike rate, economy, etc.)
- Team-wise analysis (best/worst player per team, total runs/wickets per team)
- Top-N rankings (run scorers, wicket takers, strike rate, economy, etc.)
- Head-to-head player comparisons
- Best all-rounder (combined batting + bowling score)
- Visualizations: bar charts, histograms, scatter plots
- Simple menu-driven CLI to explore everything

## Files
- `batter_code.py` – batting stats and analysis functions
- `bowler_code.py` – bowling stats and analysis functions
- `main.py` – menu-driven entry point tying both together
- `IPL2025Batters.csv`, `IPL2025Bowlers.csv` – dataset (source: Kaggle)

## Tech Stack
Python, pandas, numpy, matplotlib

Dataset sourced from Kaggle. This is a personal learning project for data analysis practice.
